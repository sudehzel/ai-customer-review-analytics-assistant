💬 AI-Powered Customer Review Analytics Assistant

An AI-powered customer review analytics platform that transforms thousands of customer reviews into actionable business insights using Natural Language Processing (NLP), Large Language Models (LLMs), and interactive dashboards.

⸻

🚀 Project Overview

AI-Powered Customer Review Analytics Assistant enables businesses to upload customer review datasets and automatically analyze customer sentiment, complaint categories, key discussion topics, and business improvement opportunities.

The platform combines traditional NLP techniques with Gemini LLM capabilities to generate intelligent summaries, actionable recommendations, and interactive question-answering experiences.

⸻

✨ Features

📊 Review Analytics

* Upload customer review datasets in Excel format
* Automatic sentiment classification (Positive / Neutral / Negative)
* Customer complaint category detection
* Most frequent keyword extraction
* Negative review identification

📈 Interactive Dashboard

* KPI cards
* Sentiment distribution analysis
* Complaint category distribution
* Plotly donut charts
* Word Cloud visualization
* Interactive filtering options

🤖 AI-Powered Insights

* Gemini-powered business summary generation
* AI Action Plan recommendations
* Natural language question answering
* Conversational review analysis with chat history

📄 Reporting

* PDF report generation
* Downloadable analytics summaries
